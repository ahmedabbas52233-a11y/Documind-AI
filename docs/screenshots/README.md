# Screenshots

Add these three screenshots to this folder:

| Filename | What to capture |
|---|---|
| `upload.png` | The upload page with drop zone visible |
| `analysis.png` | Analysis results page showing summary + key points |
| `history.png` | History page with a few documents listed |

**How to take them:**
1. Run the app locally (`npm run dev` + `uvicorn`)
2. Use browser DevTools → Toggle device toolbar → set to 1440×900
3. Screenshot each page and save to this folder
4. Commit and push — the README will automatically show them

**Recommended tool:** [Lightshot](https://app.prntscr.com/) (free, Windows/Mac)  
or Chrome DevTools → More tools → Capture screenshot
